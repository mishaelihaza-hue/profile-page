
# Portfolio Website

A simple, responsive portfolio website built with HTML, CSS, and Bootstrap. It includes a home page with a hero carousel, skills, and projects, plus About and Contact pages.

## Pages

- `index.html`: Home page with hero carousel, skills, and projects
- `about.html`: About section
- `contact.html`: Contact form

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Running Locally

Open `index.html` in a browser.

## Notes

- The hero and project images are linked from Unsplash in `index.html`.
- If you prefer local images, place them in `images/` and update the `src` paths.
